# Petplus Mobile App - Detailed Wireframes

## Design System

### Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#2E7D32` | Buttons, icons, active states, navigation |
| Primary Light | `#4CAF50` | Hover states, progress indicators |
| Primary Dark | `#1B5E20` | Text on light backgrounds |
| Secondary | `#FF8F00` | Warnings, highlights, badges |
| Secondary Light | `#FFF3E0` | Accent backgrounds, tags |
| Background | `#FFFFFF` | Main screen background |
| Surface | `#F5F5F5` | Cards, input backgrounds |
| Surface Elevated | `#FFFFFF` | Cards with shadow |
| Text Primary | `#212121` | Headings, body text |
| Text Secondary | `#757575` | Descriptions, placeholders |
| Text Disabled | `#BDBDBD` | Inactive states |
| Error | `#D32F2F` | Errors, delete actions |
| Success | `#1B5E20` | Success states, confirmations |
| Warning | `#F57C00` | Warnings, alerts |
| Border | `#E0E0E0` | Dividers, input borders |
| Shadow | `rgba(0,0,0,0.1)` | Card shadows |

### Typography
| Style | Size | Weight | Usage |
|-------|------|--------|-------|
| H1 | 24px | Bold | Screen titles |
| H2 | 20px | Bold | Section headers |
| H3 | 18px | Semi-bold | Card titles |
| Body | 16px | Regular | Main text |
| Body Small | 14px | Regular | Descriptions |
| Caption | 12px | Regular | Labels, timestamps |
| Button | 16px | Semi-bold | Button text |

### Spacing
- Base unit: 8px
- Small: 8px
- Medium: 16px
- Large: 24px
- XLarge: 32px
- Card padding: 16px
- Screen padding: 16px
- Border radius: 12px (cards, buttons), 8px (inputs, small elements)
- Shadow: `0 2px 8px rgba(0,0,0,0.1)` (cards)

### Component Styles

**Buttons:**
- Primary: Background `#2E7D32`, text white, border-radius 12px, height 48px
- Secondary: Background `#FFF3E0`, text `#2E7D32`, border 1px solid `#2E7D32`
- Disabled: Background `#E0E0E0`, text `#BDBDBD`
- Icon: 48px circle, `#2E7D32` background, white icon

**Cards:**
- Background white, border-radius 12px
- Shadow: `0 2px 8px rgba(0,0,0,0.1)`
- Padding: 16px

**Inputs:**
- Background `#F5F5F5`, border-radius 8px
- Height: 48px
- Padding: 12px 16px
- Focus: Border 2px solid `#2E7D32`
- Error: Border 2px solid `#D32F2F`

**Bottom Navigation:**
- Height: 64px
- Background white
- Border-top: 1px solid `#E0E0E0`
- Active: Icon `#2E7D32`, label `#2E7D32`
- Inactive: Icon `#757575`, label `#757575`
